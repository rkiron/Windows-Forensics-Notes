
# File & Data Recovery 


## Objective
Recover and check deleted files 

## Inputs
- File/memory: Disk Image
- Tool: Autopsy (in this case)


## Core Workflow

1. Open Autopsy 

2. Setup the new case. 

3. Select the appropiate data source type (depends on what you want to analyse), location of data source (the disk image in this case), modules to be used (if any)

4.  Contents of the disk are shown on right side upper tab. Deleted side files/folder have X marked on them. 

5. Right click and extract them to recover. Provide path for the recover file to get saved in
